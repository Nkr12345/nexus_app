## HOW TO RUN ON YOUR ANDROID PHONE

### Option 1 — Host Free on GitHub Pages (Recommended)
1. Go to github.com — create free account
2. Create new repository called "nexus-app"
3. Upload all 4 files: index.html, sw.js, manifest.json, README.md
4. Go to Settings → Pages → Source: main branch → Save
5. Your app is live at: https://YOUR-USERNAME.github.io/nexus-app
6. Open that link in Chrome on your Android phone
7. Chrome shows "Add to Home Screen" — tap Install!

### Option 2 — Host Free on Netlify
1. Go to netlify.com
2. Drag the entire nexus-app folder onto the page
3. Done — you get a live URL instantly
4. Open in Chrome on Android → Install to home screen
